Definição de Sistemas: Sistemas são um conjunto de elementos que interagem para a realização de uma 
                       série de objetivos de modo organizado e coordenado, como por exemplo: 
   Automóvel - transportar passageiros e carga - subistemas: motor, caixa de marchas, suspensão;
   Sistema digestivo
   Sistema solar
   Computador

Definição de Sistemas de Informação:
	Sabendo-se o que são sistemas de uma maneira genérica, como seria então os chamados
	"Sistemas de Informação", que da nome ao curso? Para compreender melhor o que são,
	precisamos saber a definição clara de: DADOS e INFORMAÇÃO

	      DADO: 23 anos
	INFORMAÇÃO: 23 anos de idade

	DADOS: É uma cadeia de caracteres ou padrões sem interpretação. É qualquer elemento
	       identificado em sua forma bruta que por si só não conduz a uma compreensão
	       de determinado fato ou situação, como por exemplo: nome de um funcionário;

	INFORMAÇÃO: É um conjunto de dados aos quais seres humanos deram forma para torná-los
               fignificativos e úteis. Dados dotado de relevância: quantidade de vendas 
               por produto, total de vendas mensais, numero de votos, etc.

	EXEMPLOS: 5000 crianças é um dado;
	          5000 crianças foram vacinadas no ano de 2015 é uma informação.

	          Coleta de dados para uma pesquisa eleitoral;
	          Resultado consolidade ou processado da coleta e informação (qual candidato que teve mais votos)

Conhecimento: Capacidade de resolver problemas, inover e aprender baseando-se em experiÊncias prévias;
              Esforço de investigação para descobrir aquilo que está oculto, que não esta compreendido ainda;
              Adquirir conhecimento não é reter informação, mas utilizar esta para desvendar o novo e avançar.

              INFORMAÇÕES tornam-se CONHECIMENTO quando integradas com outras informações em uma forma útil
              para tomar decisões e determinar ações.

Aula 2016-03-08

Sistema: É um conjunto ou combinações de coisas ou partes, formando um todo complexo ou unitário.

Os sistemas existem dentro de sistemas: Cada sistema é constituído de subistemas e, ao mesmo, faz parte de um sistema maior.

Parâmetros de um Sistema:

				 Entrada ou insumo -> É a força ou impulso de arranque ou de partida do sistema que fonece o material ou energia ou
						 			  informação para a operação do sistema;

 	 			  Saída ou Produto -> É a consequência para a qual se reuniram elementos e relações do sistema;

	Transformação ou Processamento -> É o fenômeno que produz mudanças, ou seja, é o mecanismo de conversão das entradas em saídas;

			  Retroação (Feedback) -> Pode ser considerado como a reintrodução de uma saída sob a forma de informação. 
			  						  É uma regulação retroatica desencadeada por uma nova informação, a qual afetará seu 
			  						  comportamento subsequente;

		    Controles e avaliações -> Principalmente para verificar se as saídas estão coerentes com os objetivos estabelecidos. 
		    						  Para controlar e avaliar de maneira adequada é ncessária uma medida de desempenho do sistema, 
		    						  chamada padrão.

Tipos de Sistemas:

	Sistemas Fechados: São os sistemas que não apresentam intercâmbio com o meio ambiente que os circunda, pois são herméticos a
					   qualquer influência ambiental;

   	 Sistemas Abertos: São os sitemas que apresentam intercâmbio com o ambiente através de entradas e saídas. Os sitemas abertos
   	 				   trocam matéria e energia com o meio ambiente;

Aula 2016-03-22

Responder as mesma 9 perguntas para um sistema de uma pizzaria.

Sistema: Software para controle de Pizzaria (offline: pedidos via telefone ou presencial)

1. Por que ele é a junção de várias funções (cadastro de pedidos, estoque, etc.) em um único lugar. O software recebe os 
   dados, armazena e processa as informações e com isso fornece controle para o funcionário.

2. O software trabalha com informãções, e essas informações são importadas pelo usuário quando um cliente faz um pedido, 
   ou quando o funcionário controla o estoque de produtos, etc.

3. Dados que geram informações, porém, de maneira orgazinada e que o usuário possa controlar toda a sua empresa 
   (Pizzaria). Por exemplo: O usuário cadastra todo seu estoque no sistema, e o mesmo organiza essas informações amigavelmente
   para o úsuário ter uma visão melhor de como está seu estoque.

4. O software trabalha todas suas informações exibindo organizadamente em telas separadas para o usuário, cada uma com 
   seu assunto e objetivo.

5. Através de manutenção de quem desenvolveu o software, para acompanhar o avanço da tecnologia e correções de problemas
   internos (bugs) do sistema.

6. O sistema informa toda e qualquer operação realizada pelo usuário, seja ela certa ou errada, através de balões 
   informativos, mensagens e avisos. EX: Ao se cadastrar um produto o sistema exibe uma mensagem para o usuário de que 
   foi "Cadastrado com Sucesso", ou no caso de alguma falha, uma mensagem amigável mostrando o motivo do erro.

7. Para um software se manter estável, é necessário o mesmo estar hospedado em um servidor que aguente a carga de usuários
   e informações importadas para o sistema.

8. Possuindo qualidade, uma interface agradável, sem falhas (bugs), de fácil uso e de manunteção de baixo custo para o usuário.

9. Depende, em um software, cada parte dele é definido para efetuar uma função expecífica, sendo inteligente e objetivo, porém,
   essas funções podem ser executadas de várias maneiras pelo usuário.
